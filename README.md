# 📊 Análisis Competitivo Restaurantes BMV — 3T2025
### Alsea vs Grupo Gigante Restaurantes | Oportunidad Mundial 2026

## 🎯 Objetivo
Identificar el GAP competitivo entre Alsea y Gigante Restaurantes 
usando datos reales de reportes trimestrales BMV, contexto 
macroeconómico y calendario deportivo como drivers de análisis.

## 🔍 Hallazgos principales
- GAP de **240 BPS** en VMT: Alsea +4.10% vs Gigante +1.70%
- Gigante creció en ingresos (+3.9%) pero utilidad operativa cayó -12.3%
- Alsea captura **37.4%** de ventas por canales digitales
- Proyección Mundial 2026: Gigante podría alcanzar VMT **4.20%**
